drift — Quantify your focus loss

A terminal-native tool that tracks your active window, detects context switches, and shows exactly how much your attention is drifting.

$cargo install drift-tracker

Why?

23 minutes

That's how long it takes to refocus after a context switch. Yet developers switch between code, Slack, Twitter, and email hundreds of times per day.

Source: Gloria Mark, UC Irvine — "The Cost of Interrupted Work" (2008)

Features

Background Tracking

Monitors your active window every 2 seconds. Categorizes apps into code, distraction, communication, research, and system.

Focus Score

0–100 score based on switch count, distraction time, and focus loss. Know exactly how your day went.

TUI Dashboard

Real-time terminal UI with category breakdowns, switch history, and live focus streaks.

Focus Mode

Start timed focus sessions. Track interruptions and measure how long you actually stay in the zone.

Insights & Compare

Discover your best focus hours, most distracting apps, and compare days or weeks side-by-side.

Shell Completions

Generate completions for bash, zsh, fish, and PowerShell. Tab-complete every command and flag.

Export & Reports

Daily and weekly reports with trend analysis. Export to JSON or CSV for your own pipelines.

Cross-Platform

Works on Linux (X11), macOS, and Windows. No cloud, no accounts — your data stays local.

Usage

Getting started

# Install
$ cargo install drift-tracker

# Initialize config
$ drift init

# Apply a preset
$ drift preset development

# Start tracking in the background
$ drift track

Reports & Insights

# Daily report
$ drift report

# Weekly report with trend
$ drift week

# Smart insights from last 7 days
$ drift insights

# Compare two days
$ drift compare --date1=2026-07-20 --date2=2026-07-22

# Compare this week vs last week
$ drift compare --week

Focus & Dashboard

# Start a 90-minute focus session
$ drift focus 90

# Open the TUI dashboard
$ drift show

# Live watch mode
$ drift watch

# Export data as JSON
$ drift export --format=json

Shell Completions

# Bash
$ drift completions bash >> ~/.bashrc

# Zsh
$ drift completions zsh > ~/.zfunc/_drift

# Fish
$ drift completions fish > ~/.config/fish/completions/drift.fish

# PowerShell
$ drift completions powershell | Out-String | Invoke-Expression
